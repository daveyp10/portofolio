AI-Generated Carousel Pipeline

Problem
A client with 4 brand accounts across Instagram, TikTok, YouTube, and Facebook needed consistent multi-platform posting 3× daily — with no way to share primary Instagram credentials through third-party APIs.
Solution
  • Content generation — DeepSeek API writes quote/psychology content per brand voice, formatted for carousel structure.
  • Slide rendering — Pillow generates carousel images with dynamic text overlays; orphan-word logic ensures clean typography on every slide.
  • Distribution — Vista Social API handles multi-platform scheduling. A dedicated Android emulator handles Instagram natively for music injection, keeping credentials off third-party servers.
  • Infrastructure — Runs 24/7 on a Linux VPS via nohup, tracked via Google Sheets.

AI Video Production Pipeline

Problem
Multi-step manual video production — scripting, voice recording, b-roll sourcing, editing — needed to collapse into a single command that outputs broadcast-ready short-form videos at scale.
Solution
  • Scripting — DeepSeek API generates psychology/deep-talk scripts formatted for voiceover and caption burn-in.
  • Visuals — Google Veo (via Gemini API) generates cinematic background video matched to the script's mood and pacing.
  • Voice — ElevenLabs API renders emotionally nuanced voiceovers; preferred over OpenAI TTS for expressive quality on this content style.
  • Assembly — FFmpeg compiles all layers and burns frame-accurate subtitles synced to the millisecond.
Stack
Python DeepSeek API Google Veo / Gemini API ElevenLabs API FFmpeg

Problem

Solution
